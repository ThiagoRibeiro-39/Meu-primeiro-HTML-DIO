Inspetor de Elementos

(Voltar)

O "Inspetor de Elementos" é uma ferramenta poderosa disponível nos navegadores da web que permite aos desenvolvedores examinar e depurar o código HTML, CSS e JavaScript de uma página.

Exame de Estrutura HTML e CSS: Visualização da árvore de elementos HTML da página. Seleção e realce de elementos específicos na página ao passar o mouse sobre o código-fonte. Exibição das regras CSS aplicadas a um elemento, incluindo propriedades, valores e a origem da regra.

Edição ao Vivo: Edição instantânea de código HTML e CSS diretamente no navegador. Visualização das alterações em tempo real na página.

Depuração JavaScript: Visualização e definição de pontos de interrupção (breakpoints) no código JavaScript. Inspeção de variáveis, expressões e pilha de chamadas durante a execução do código.

Rede e Desempenho: Monitoramento das solicitações e respostas da rede (HTTP). Análise de desempenho da página, incluindo tempo de carregamento e recursos consumidos.

Console JavaScript: Execução de comandos JavaScript no console. Exibição de mensagens de erro, aviso e log geradas pelo código JavaScript.

Auditorias e Testes de Acessibilidade: Ferramentas integradas para realizar auditorias de desempenho, acessibilidade e SEO.

Emulação de Dispositivos: Simulação de diferentes dispositivos e tamanhos de tela para testar a responsividade do design.

A utilização do Inspetor de Elementos é fundamental para desenvolvedores web durante o processo de desenvolvimento, pois oferece insights detalhados sobre a estrutura e o comportamento de uma página da web, além de facilitar a resolução de problemas e a otimização de código. Para acessar o Inspetor de Elementos, normalmente você pode clicar com o botão direito em qualquer parte de uma página da web e selecionar "Inspecionar" ou usar o atalho de teclado, como Ctrl+Shift+I no Windows ou Cmd+Opt+I no macOS.

Estrutura básica do HTML

(Voltar)

Os arquivos HTML se subdividem em "Head" e "Body". A estrutura básica do HTML é composta por elementos essenciais que fornecem a base para a construção de uma página web.
Aqui está um exemplo simples da estrutura básica do HTML:

DOCTYPE html: Declaração do tipo de documento, indicando que a página segue a especificação do HTML5.

html lang="en": Elemento raiz que envolve todo o conteúdo da página. O atributo lang especifica o idioma da página (neste caso, inglês).

head: Contém metadados, como o conjunto de caracteres (meta charset="UTF-8"), a configuração de viewport para dispositivos móveis e o título da página (title).

body: Contém todo o conteúdo visível da página, como cabeçalhos, parágrafos, imagens, etc.

Comentários (< !-- ... -- >): São usados para adicionar notas ou explicar partes do código. Não são exibidos na página renderizada.

Essa estrutura fornece um ponto de partida para a construção de páginas web. Dentro do body, você pode adicionar elementos HTML adicionais para criar o layout desejado e incluir conteúdo específico.

Falando sobre Tags

(Voltar)

As tags HTML são os blocos de construção fundamentais para criar páginas web. Elas são elementos cercados por colchetes angulares (< >) e geralmente ocorrem em pares de abertura e fechamento. Aqui estão algumas tags HTML comuns e suas funções.

As tags em HTML (Hypertext Markup Language) são elementos fundamentais que estruturam e definem o conteúdo de uma página web. Cada tag tem uma função específica e ajuda a organizar o conteúdo para que os navegadores possam interpretar e exibir corretamente.

Atributos das Tags

(Voltar)

Os atributos são propriedades que as tags podem ter em HTML podem ser divididos em algumas categorias principais, dependendo de sua função e onde podem ser usados. Aqui estão algumas categorias comuns de atributos:

Atributos Globais: São aqueles que podem ser usados em quase todos os elementos HTML. Exemplos incluem class, id, style, title, lang, dir, etc.

Atributos de Evento: Utilizados para associar scripts a eventos específicos. Exemplos incluem onclick, onmouseover, onchange, etc.

Atributos de Controle de Formulário: Usados em elementos de formulário para controlar seu comportamento. Exemplos incluem name, value, disabled, readonly, placeholder, etc.

Atributos de Links e Navegação: Específicos para elementos relacionados à navegação e links. Exemplos incluem href, target, rel, download, etc.

Atributos de Mídia: Relacionados a elementos de mídia, como imagens, áudio e vídeo. Exemplos incluem src, alt, width, height, autoplay, etc.

Atributos de Tabelas: Usados em elementos de tabela para controlar sua estrutura e apresentação. Exemplos incluem border, cellpadding, colspan, rowspan, etc.

Atributos de Estilo e Layout: Utilizados para controlar a apresentação e o estilo. Exemplos incluem style, class, width, height, etc.

Atributos de Scripts: Utilizados em elementos de script para controlar seu comportamento. Exemplos incluem async, defer, type, etc.
Essas categorias são apenas uma visão geral e muitos atributos específicos são exclusivos para determinados elementos.

Textos (tipografia)

(Voltar)

Tipografia no HTML refere-se ao modo como o texto é apresentado em uma página web. A escolha de fontes, tamanhos, estilos e cores de texto desempenha um papel importante na legibilidade e na estética de uma página.

Elementos de Texto: A letra 'p' entre os sinais de maior que e menor que define um parágrafo de texto. 'h1', 'h2', ..., 'h6' entre os sinais de maior que e menor que define cabeçalhos de diferentes níveis, geralmente usados para títulos e subtítulos. 'span' também entre os sinais maior que e menor que pode ser usado para envolver pequenas partes de texto para aplicar estilos específicos.

Atributos de Texto: style: Pode ser usado para aplicar estilos diretamente a um elemento de texto.

Fontes Personalizadas: Podemos usar fontes personalizadas em suas páginas web com a propriedade font-family. Certifique-se de incluir a referência correta à fonte no seu CSS.

Unidades de Tamanho: px (pixels): Unidade fixa, boa para controle preciso do tamanho do texto. em: Relativo ao tamanho do texto no elemento pai. %: Porcentagem do tamanho do texto no elemento pai.

Cores de Texto: color: Define a cor do texto.

Estilos de Texto: font-style: Pode ser normal, italic ou oblique para definir o estilo do texto.

Peso da Fonte: font-weight: Pode ser usado para especificar a espessura da fonte (normal, bold, bolder, lighter ou um número entre 100 e 900).

Alinhamento de Texto: text-align: Controla o alinhamento horizontal do texto (left, right, center, justify).

Listas no HTML

(Voltar)

No HTML, as listas são usadas para organizar e estruturar informações. Existem três tipos principais de listas: listas ordenadas (ol), listas não ordenadas (ul) e listas de descrição (dl).

Listas Ordenadas (ol) Uma lista ordenada é usada quando a ordem dos itens é significativa. Cada item é precedido por um número ou uma letra. Os itens são definidos pelos elementos 'li' (item de lista).

Listas Não Ordenadas (ul) Uma lista não ordenada é usada quando a ordem dos itens não é significativa. Os itens são marcados com bullets (pontos, discos, quadrados, etc.), e cada item é definido pelos elementos 'li'.

Listas de Descrição (dl) Uma lista de descrição é usada para associar termos a suas definições. Cada definição é precedida por um termo (dt) e seguida pela descrição (dd).

Atributos Importantes: type (para ol): Define o tipo de marcador usado na lista ordenada (1, A, a, I, i). start (para 'ol'): Especifica o número inicial da lista ordenada. reversed (para <'ol'): Inverte a ordem dos itens da lista ordenada.

Aninhamento de Listas: Podemos aninhar listas dentro de outras listas para criar estruturas mais complexas. Esses são os conceitos básicos relacionados a listas em HTML. Elas são ferramentas essenciais para organizar e apresentar informações de maneira estruturada em páginas web.
